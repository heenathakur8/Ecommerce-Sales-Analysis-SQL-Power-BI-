# 📊 E-Commerce Sales & Customer Behavior Analysis

An interactive data analytics project focused on understanding **sales performance, customer behavior, and business inefficiencies** using Power BI.

---

## 🚀 Project Overview

This project analyzes an e-commerce dataset to identify:
- Revenue trends and growth patterns  
- Customer conversion and drop-off points  
- Product category performance  
- Return behavior and regional insights  

The goal is to uncover **actionable insights** that can help improve revenue and customer experience.

---

## 📈 Key Insights

- 📉 Revenue declined by **8.98% YoY**
- 🛒 Conversion Rate is **7.15%**
- ⚠️ Cart Abandonment Rate is **66.71%**
- 🔁 Return Rate is **44.01%**
- 🏆 **Electronics** is the top revenue-generating category  

👉 Major drop observed at **cart → purchase stage**, indicating potential checkout friction.

## 🧠 Business Insights

- Revenue decline driven by **low conversion and high abandonment**
- High return rate impacts overall profitability  
- Sales are concentrated in a few categories and regions  
- Customer journey shows inefficiencies across multiple stages  

---

## 💡 Recommendations

- Improve checkout experience to increase conversions  
- Reduce returns by improving product quality & descriptions  
- Focus on high-performing categories  
- Expand into underperforming regions  
- Implement customer retention strategies
